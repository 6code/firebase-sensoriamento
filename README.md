# Sensoriamento Realtime com Firebase e ESP8266

#### BOM
* Qualquer MCU com ESP8266 ( Foi usado um wemos d1 mini)
* Sensor de Presença PIR
* Sensor de temperatura e umidade DHT11 ou DHT22
* Led 
* Resitor 200 ohm
